# Nexus Sidebar v1.0.0

The ultimate productivity sidebar for Opera GX & Chrome.

## Install (Developer Mode)
1. chrome://extensions/ → Enable Developer mode
2. Load unpacked → select nexus-sidebar folder
3. Click the Nexus icon in your toolbar

## Key Shortcuts
- Ctrl+Shift+S — toggle sidebar
- Ctrl+Shift+T — open Todo
- Ctrl+Shift+N — open Notes
- Ctrl+Shift+P — open Pomodoro
- Ctrl+K — global search

## Pinning Panels
Click ⋯ More in the nav → use 📍 to pin/unpin panels. Max 7 pinned at once.
