# Nexus V0

This directory is a conflict-reduced branch snapshot of the extension.

## Why this exists
- Isolates the branch state into a fresh folder (`Nexus-V0/`) so you can open PRs against this folder directly.
- Avoids touching prior paths during review, which helps when there are many merge conflicts.

## Contents
- `nexus-sidebar/` (full extension source)
- `CHANGELOG.md`

## Quick start
1. Open extensions page in your browser.
2. Enable Developer Mode.
3. Load unpacked extension from:
   - `Nexus-V0/nexus-sidebar/`

## Version note
This is a V0 snapshot branch layout for easier conflict handling.
